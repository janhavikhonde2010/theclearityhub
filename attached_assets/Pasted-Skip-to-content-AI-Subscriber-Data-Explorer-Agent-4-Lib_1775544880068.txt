Analysis Dashboard Requirement (Detailed)
----------------------------------------

OBJECTIVE: 
Create an Analysis Dashboard to evaluate lead activity, agent performance, and message sequence effectiveness.

----------------------------------------

KEY DASHBOARD METRICS

1. Agent-wise Data
- Lead history (Date & Time-wise)
- Label-wise segmentation

2. Message Sequence Analysis
- Total sequence messages sent
- Number of leads reactivated after sequence

3. Best Performing Sequence
- Identify which sequence generates the highest reactivation rate

4. User Reply vs Wise Parrot Reply
- Compare user messages and bot replies (Wise Parrot + Sequence messages)

5. Label-wise Bar Chart
- Distribution of leads based on labels

6. Dormant Leads Count
- Leads with no reply from the user side

7. Lead Data Details
- Lead Name: message_content → contacts → profile → name
- Phone Number: contacts → wa_id
- Assignment Timeline: system messages containing "Conversation was assigned to..."
- Current Assignee: Last assignment in the timeline
- Assigned Sequence: e.g., Generic_Seq_Msg
- Last User Activity: Last message date & time

----------------------------------------

SYSTEM WORKFLOW

Step 1: Fetch Subscriber Data
Call Subscriber List API to retrieve:
- User Name
- User Phone Number
- Label Name
- Last Message Time

Step 2: Fetch Conversation Data
Using Phone Number, call Get Conversation API to retrieve:
- Assigned Agent
- Assigned Sequence
- Assignment History
- Last User Message
- User Reply Count
- Wise Parrot Reply Count
- Post-sequence replies
- Agent-wise lead assignment

Step 3: User Input
- Account API Key
- Phone Number ID

Step 4: Backend Processing
- Call APIs using credentials
- Process data
- Generate dashboard insights

----------------------------------------

DASHBOARD OUTPUTS
- Agent-wise lead analysis
- Message sequence performance
- Best performing sequence
- User vs Bot reply comparison
- Label-wise bar chart
- Dormant leads count
- Detailed lead information

----------------------------------------

SUMMARY
This dashboard provides a complete analytical view of:
- Lead engagement
- Agent performance
- Message sequence effectiveness
- User interaction behavior
